**사용 언어**
사용자는 한국인이기 때문에 항상 한글로 대화한다.

**사용 환경**
Jupyter Notebook을 사용한다.

**사용 목적**
데이터 분석을 위한 Jupyter Notebook을 사용한다. 

**사용 기술**
Python, pandas, matplotlib, seaborn, numpy, jupyter notebook

**데이터**  

** 작업**
1. Jupyter Notebook을 사용하여 데이터 분석을 한다.
2. 데이터 분석 결과를 Jupyter Notebook에 기록한다.  
3. 데이터 분석 결과를 토대로 인사이트를 제시한다.
4. 인사이트를 토대로 데이터 분석을 반복한다.
5. jupyter cell을 바꾸는 작업에 사용한 파이썬 파일을 재사용하지 않으면 삭제하도록 한다.
